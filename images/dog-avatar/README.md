# 西高地白梗卡通图案 (West Highland White Terrier Cartoon Avatars)

为狗狗军师项目生成的卡通风格西高地白梗图案。

## 文件说明

| 文件名 | 描述 | 用途 |
|--------|------|------|
| `westie-happy.svg` | 快乐表情 | 首页/排行榜 |
| `westie-cute.svg` | 可爱大眼 | 排行榜 |
| `westie-wink.svg` | 眨眼表情 | 排行榜 |
| `westie-playful.svg` | 戴蝴蝶结 | Tab icon |
| `westie-star.svg` | 星星配饰 | Tab icon |

## 使用说明

1. **SVG 转 PNG**: 使用在线工具或设计软件将 SVG 转换为 PNG
   - 推荐尺寸: 512x512px (头像), 200x200px (Tab icon)
   - 确保保留透明背景

2. **设计软件**:
   - Figma: 导入 → 导出 PNG
   - Sketch/Illustrator: 打开 → 导出

## 设计特点

- 卡通/可爱风格 (kawaii style)
- 白色毛发 + 米色/奶油色背景
- 西高地白梗标志性立耳
- 友好、活泼的表情
- 圆润的大眼睛 + 黑色鼻子
