## 🔗 页面跳转

```
index (首页)
  ├── [🔮 生成] → result (结果页)
  └── [💬 热门] → hot (热门问答)

result (结果页)
  ├── [🔄 再问] → index
  └── [← 返回] → index

history (历史记录)
  └── [点击] → result
```
